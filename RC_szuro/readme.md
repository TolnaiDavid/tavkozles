# RC szűrő kapcsolás(Alul áteresztő)    

## Születési idő: 2006.07.26    

$R= 7ohm$     
$C= 26uF$      
---  
Kiszámolom a határ frekvenciát:      
$$fh = {1 \over (2 * π * R * C)}$$   
$$fh = {1 \over (2 * π * 26 uF * 7 ohm)}=874,4 Hz$$
